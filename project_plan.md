# Aditya Sawant Developer Portfolio - Project Plan

A modern, fast, and SEO-friendly single-page React portfolio for a Backend / Fullstack Software Engineer.

## Goal Description
Build a clean, minimal, and professional developer portfolio for Aditya Sawant that effectively showcases their 2 years of backend (Java/Spring Boot) and architecture experience along with projects and achievements.

## Technology Stack
- **Framework**: React with Vite
- **Styling**: Vanilla CSS (CSS Modules or global index.css with modern CSS architecture, custom properties)
- **Deployment**: Configured for static hosting

## Proposed Architecture

### Core Foundation
We will start by configuring a React app using Vite and integrating a robust, premium visual design system using vanilla CSS (variables for colors, modern typography, spacing, and micro-animations).

### Components Structure
- **Layout**: `Navbar.jsx`, `Footer.jsx`
- **Sections**:
  - `HeroSection.jsx`: Welcome text + quick action buttons
  - `AboutSection.jsx`: Professional summary
  - `TechStackSection.jsx`: Grouped list of technologies
  - `WorkExperienceSection.jsx`: Professional timeline
  - `ProjectsSection.jsx`: Grid cards showcasing projects
  - `SystemDesignSection.jsx`: Architecture explanations/diagrams (placeholder structure)
  - `AchievementsSection.jsx`: Notable highlights
  - `ContactSection.jsx`: Simple contact layout
